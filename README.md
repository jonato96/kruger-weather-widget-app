## Getting Started

**In this repository I will practice React with the weather-widget app.**

## Run it

```bash
# Clone Repository
$ git clone https://github.com/jonato96/kruger-weather-widget-app
# Go to server folder
$ cd kruger-weather-widget-app
# Install Dependencies
$ npm i
# Run Aplication
$ npm start
```
